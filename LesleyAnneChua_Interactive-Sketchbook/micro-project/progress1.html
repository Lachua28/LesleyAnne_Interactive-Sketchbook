Micro-Project: Progress #1

1. Concept Summary

The interaction is that the user can click in order to board the bus as the bus drives to the bus stop. The user controls when they attempt to board the bus, which will only be successful at a certain time/location. The change over time is that the bus moves automatically, but stops whenever the user attempts to board the bus.

2. Primary Variable

Main variable: busX

This variable affects the x-position of the bus and is how the system determines whether you can successfuly board the bus. Success is when busX is within a certain range.

3. Base Build (What You Will Code This Week)

For this week, I wanted to see if I could teach myself the automated movement over time across the screen! I've already figured out the basic functions, so this week I plan to add more visuals to differentiate the state changes and where you can board the bus. I'm also going to add a little person to show the bus has been boarded.

State 1: The bus is empty and begins to move towards the bus stop.
State 2: If you attempt to board the bus before it is at the stop, the system stops the bus and tells you that you can't board.
State 3: If you attempt to board the bus when it's at the stop, the system stops the bus and tells you you've succeeded.
